# Olist E-Commerce Data Analysis

## Project Overview
This project analyzes the Brazilian Olist e-commerce dataset using SQL and Python.

## Tools Used
- PostgreSQL
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Brazilian E-Commerce Public Dataset by Olist (Kaggle)

## Key Analysis
- Order status distribution
- Payment method distribution
- Top product categories
- Revenue by payment type
- Delivery time analysis

## Visualizations
Charts are stored in the `visualizations` folder.

## Key Insights
- Most orders are successfully delivered.
- Credit cards are the most common payment method.
- Some product categories dominate the marketplace.
